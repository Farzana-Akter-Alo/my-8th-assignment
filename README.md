# Hero.Io

## Description

**Hero.Io** built with React.  
It allows users to browse apps, search, view app details, install apps, and manage installed apps.  
The app includes interactive features such as live search, installation management with LocalStorage, and a responsive chart for app reviews.

The UI is designed following a Figma reference, including a header, footer, home page, all apps page, app details page, and my installation page.  
It also supports smooth navigation, and error handling for invalid routes.

---

## Technologies Used

- **React** (with React Router v6)
- **Tailwind CSS** for styling
- **Recharts** for visualizing app review charts
- **React Toastify** for notifications
- **LocalStorage** for storing installed apps
- **Deployment** on Surge
